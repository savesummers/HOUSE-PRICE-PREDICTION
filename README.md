# 🏠 House Price Prediction using Linear Regression

This project implements a simple **House Price Prediction model** using **Linear Regression** from `scikit-learn`. It analyzes how different housing features influence property prices and allows users to input custom values to get predicted prices.



## 📊 Features Used

The model is trained on the following variables:

- Average Area Income  
- House Age  
- Number of Rooms  
- Number of Bedrooms  
- Area Population  



## 🔍 Exploratory Data Analysis

Scatter plots are generated to visualize the relationship between each feature and house prices.  
This helps in understanding feature importance.

> 📌 Insight: *Number of Bedrooms is not a strong predictor compared to other features.*



## 🤖 Model

- Algorithm: **Linear Regression**
- Library: `sklearn.linear_model`
- Multi-feature regression approach
